This question bank for **CO1 — Introduction & AI Search Algorithms** is compiled based on an analysis of the four provided semester papers (June 2024, June 2025, Aug/Sept 201x, and Aug/Sept 2024).

### **Part A: Knowledge & Definitions (3 Marks)**

1. **Define Artificial Intelligence** as stated by John McCarthy.
2. Highlight the purpose and the process followed in the **Turing Test**.
3. Define the **PAGE factors** involved in designing a task environment for an agent.
4. Identify and describe the role of the **labeled parts (Sensors/Actuators)** in the functional flow of an intelligent agent.

### **Part B: Conceptual & Intermediate Tracing (7-8 Marks)**

1. **Classical Search Tracing (DFS):** Trace the Depth First Search (DFS) for a given state space model. Clearly provide the steps involved and mention which goal state is reached first.
2. **Classical Search Tracing (BFS):** Provide the stepwise solution for a state space model using Breadth First Search (BFS) and identify the first goal state reached.
3. **Task Environment Analysis:** Provide a detailed description of the task environment (using PAGE factors) for a specific agent, such as a part-picking robot.
4. **Hill Climbing:** Examine and explain the **pitfalls of the Hill Climbing algorithm** along with their corresponding solutions.

### **Part C: Advanced Problem Solving & Tracing (10-15 Marks)**

1. __A_ Algorithm (Outline & Application):_*
    - Outline the steps of the A* search algorithm.
    - Apply the A* algorithm on a provided state space representation/graph (referencing $h(n)$ and $g(n)$ values).
    - Trace the **optimal path** to the goal state and calculate the **optimal cost**.
2. **State Space Modeling:** Explore and construct a state space model for a real-world AI problem, demonstrating the transition from a start node to multiple goal nodes using classical search.

---

### **Exam Strategy for CO1**

- **The "Guaranteed" Question:** Practice tracing the __A_ algorithm_* on various graph structures. Every single analyzed paper contains a 10 or 15-mark question requiring you to find the "optimal path" using A*.
- **The "Easy Wins" Topic:** Memorize **John McCarthy’s definition** and the **PAGE factors**. These are high-frequency, low-effort marks for Part A.
- **Tracing Precision:** When solving **DFS or BFS** questions, the examiners specifically look for "stepwise" progress. Ensure you list the fringe/queue at every step to secure full marks.
- **Agent Diagrams:** Be prepared to draw and label the structure of an agent, specifically identifying **Sensors**, **Actuators/Effectors**, and **Condition-Action rules**.
