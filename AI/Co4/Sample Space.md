Explain how a sample space is derived and represent the sample space for the following experiment: "Mani could learn a scientific or historical course in class or at home".

A **sample space** is the set of all possible outcomes of a random experiment. The process of deriving it involves identifying each independent variable or choice within the experiment and listing every unique combination of those results.

### **How a Sample Space is Derived**

To derive a sample space, you follow these logical steps:

1. **Identify the Components:** Determine the individual events, stages, or choices that make up the experiment.
2. **List Outcomes for Each Component:** Define the possible results for each specific stage (e.g., "Heads" or "Tails" for a coin).
3. **Combine Outcomes:** Use a method such as a **tree diagram** or a **Cartesian product** (pairing every item in the first set with every item in the second set) to find all possible combinations.
4. **Ensure Exhaustiveness:** The final set must include every potential result that could occur during the experiment.

---

### **Sample Space for Mani’s Experiment**


**The Components:**

- **Course Type:** {Scientific (S), Historical (H)}
- **Location:** {Class (C), Home (Ho)}

**The Sample Space ($S$):** By combining every course type with every possible location, we represent the sample space as: **$S = { (Scientific, Class), (Scientific, Home), (Historical, Class), (Historical, Home) }$**

Or in abbreviated form: **$S = { (S, C), (S, Ho), (H, C), (H, Ho) }$**

This set represents all four possible outcomes for the experiment where Mani chooses a course and a location.
