The Continuous Assessment Test 2 (CA 2), dated May 22, 2025, for the course "23MX24 - Enterprise Computing Using Full Stack," contains the following questions:

### **Question 1 (25 Marks total)**

- **Subdivision A (3 Marks):** Evaluate how the use of the **Virtual DOM** can enhance user experience and optimize rendering performance for an enterprise e-commerce platform that handles high user traffic and dynamic content updates.
- **Subdivision B (10 Marks):** For a platform called **"TravelConnect"** (where users share travel experiences, media, and use real-time messaging), evaluate the benefits and drawbacks of using a **Single Page Application (SPA)** versus a **Multi-Page Application (MPA)** for handling user profiles, media uploads, and real-time messaging.
- **Subdivision C (12 Marks):** For a web-based fitness tracking application called **"FitTrack"** (allowing workout logging, connection with coaches, and real-time progress reports), compare and contrast the use of **MEAN Stack** and **MERN Stack** Technology, considering performance, scalability, and real-time updates.

### **Question 2 (25 Marks total)**

- **Subdivision A (3 Marks):** Differentiate between **APIs and Web Services** with respect to their roles in enterprise application integration.
- **Subdivision B (10 Marks total):**
    - **i (6 Marks):** Illustrate two enterprise scenarios where **SOAP and REST** web services are used effectively and explain why each was appropriate.
    - **ii (4 Marks):** Compare how REST and SOAP web services support different enterprise application requirements, such as **payment processing** and **mobile app backend integration**.
- **Subdivision C (12 Marks total - Choose i or ii):**
    - **Option i:**
        - **a (6 Marks):** Explain how you would design, develop, deploy, and consume **Java SOAP web services** to facilitate secure data exchange for a financial enterprise system integrating with external banking services.
        - **b (6 Marks):** Analyze how the use of the web service standard stack (**SOAP, WSDL, UDDI**) ensures compatibility between systems on different platforms in a cross-enterprise supply chain application.
    - **[OR] Option ii:**
        - **a (6 Marks):** Analyze how **REST's architectural constraints** help or hinder system scalability in an enterprise application needing to handle millions of concurrent user interactions.
        - **b (6 Marks):** Explain how you would design, develop, deploy, and ensure smooth consumption of a **Restful web service** for an enterprise customer support platform, focusing on front-end and external system integration.
