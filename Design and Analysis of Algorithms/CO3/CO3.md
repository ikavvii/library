 **CO3: Apply dynamic programming to solve optimization problems**.

---

### **Part A: Theoretical Foundations & Short Numericals (5 Marks)**

1. **Principle of Optimality:** State the **Principle of Optimality**. Show that it is satisfied by the **all-pairs shortest path problem** using an example.
2. **LCS Numerical (Case 1):** Identify the **Longest Common Subsequence (LCS)** between strings $X =$ "abaaba" and $Y =$ "babbab". Show the intermediate steps.
3. **LCS Numerical (Case 2):** Given two strings $X =$ "AGGTAB" and $Y =$ "GXTXAYB", find the length of the LCS using dynamic programming.
4. **String Editing:** A senior editor needs to transform the string $X =$ "rain" to string $Y =$ "shine". Find the **minimum-cost edit sequence** that performs this transformation.
5. **Optimality Proof:** Explain how the **Longest Common Subsequence** problem holds the Principle of Optimality.

---

### **Part B: Algorithms & Application Problems (8 Marks)**

1. **All-Pairs Shortest Path (Floyd-Warshall):** Write an algorithm to find the shortest path between all pairs of vertices. Apply this to compute distances between rooms, lobbies, corridors, and stairs in a building evacuation plan.
2. **String Editing (Complex):** Find the **minimum edit distance** between the strings $X =$ "INTENTION" and $Y =$ "EXECUTION" using dynamic programming.
3. **Multistage Graph (Backward Approach):** Find the shortest path from source (S) to destination (T) using the **backward approach** for a given multistage graph. Write the corresponding algorithm.
4. **Traveling Salesman Problem (Part B variant):** A traveler plans to visit Marina Beach, Kapaleeshwarar Temple, and Fort St. George starting from a hotel. Determine the **optimal tour** and its length using a provided $4 \times 4$ adjacency matrix.

---

### **Part C: Advanced Problem Solving & Tracing (12 Marks)**

1. **Traveling Salesman Problem (TSP) Algorithm:** Write an algorithm using Dynamic Programming to determine an optimal tour. Solve for a city map starting from "Junction Hopes" with a given cost matrix.
2. **TSP Numerical (Matrix Focus):** Given a $4 \times 4$ directed graph adjacency matrix, find the **cost of the optimal tour** covering all nodes using Dynamic Programming. Show all intermediate steps.
3. **Multistage Decision Graph:** Show how dynamic programming solves the multistage decision graph problem. Find the shortest path from a source vertex (vertex 1) to a destination vertex (vertex 9) for a graph with approximately 12 edges.
4. **LCS Comprehensive:** Explain how the LCS problem is solved using dynamic programming. Given strings $X =$ "STONE" and $Y =$ "LONGEST", find the **longest common subsequence and its length**, showing the full DP table/intermediate steps.
5. **Single Source Shortest Path (DP):** Write the single-source shortest path algorithm (Bellman-Ford approach). Find the shortest distance from node S to all other nodes in a given directed graph.
