| S.No | Feature                  | Description & Relevance to System Software                                    |
|------|-------------------------|-------------------------------------------------------------------------------|
| 1    | Low-level Access        | C supports low-level access to memory via pointers, allowing direct hardware manipulation and efficient resource management, which is essential for device drivers and OS kernels. |
| 2    | Portability             | C code is highly portable across platforms with minimal modification, crucial for system software that must run on varied hardware.                      |
| 3    | Efficiency & Performance| C-generated programs have efficient use of CPU and memory; minimal runtime overhead makes it suitable for resource-constrained system environments.     |
| 4    | Simple Language Core    | C has a small set of keywords and constructs, making it easy to learn and maintain, yet powerful enough for complex system programming tasks.           |
| 5    | Rich Set of Operators   | C offers a wide variety of operators (arithmetic, bitwise, etc.), supporting fine-grained data manipulation common in system-level programming.         |
| 6    | Modularity              | C encourages breaking code into functions and files, enhancing readability, maintainability, and debugging—features critical in large system software. |
| 7    | Extensive Library Support | Although minimal itself, C provides standard libraries for essential routines as well as the facility to link with low-level system libraries.         |
| 8    | Assembly Language Integration | C can easily incorporate assembly code, allowing performance-critical parts of system software to be optimized at the lowest level.             |
| 9    | Structured Programming  | Supports structured programming, making it easier to design reliable and maintainable system software.                                                   |
| 10   | Foundation for Other Languages | Many modern system-level languages (e.g., C++, Rust) are derived from or compatible with C, demonstrating its influence and adaptability.        |
